Why Markpad

Traditional notepad speed, modern Markdown viewer.

Plain files stay plain

Open .md, .txt, .log, and other text files. No proprietary database. No import step.

Session restore

Close and reopen. Every note, draft, favorite, and active tab comes back exactly as you left it.

Markdown + Viewer

Every note has two tabs: Markdown for editing and Viewer for a clean rendered reading experience with tables, tasks, and code.

No Electron

Written in Go with Gio. A single native binary under 8 MB with low idle memory. Starts instantly.

Favorites and drafts

Star notes in the sidebar to pin them. Unsaved work survives app close through local drafts stored in your config directory.

Save anywhere

Save As lets you choose any path and any extension. Markdown, plain text, log files, or no extension at all.
